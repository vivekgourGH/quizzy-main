# Quizzy - A multi-category quiz App

Build Using Open Trivia Database.

## Tech Stack

- React.js
- JavaScript
- TailwindCSS
- Vitest (for testing)

- Wrote unit test cases for the first time.

- Learned about Snapshot testing.
